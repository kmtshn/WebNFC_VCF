WebNFC_VCF (NFC 連絡先ライター)

Webブラウザ上で連絡先情報を入力し、NFCタグへの書き込みや、vCard（.vcf：電子名刺）ファイルとしてのダウンロードができるシンプルなフロントエンドアプリケーションです。

📝 概要 (Overview)
このリポジトリは index.html のみで構成された軽量なWebアプリです。
専用のアプリをインストールすることなく、Androidスマートフォンのブラウザ（Chromeなど）から直接NFCタグ（NTAG215など）に名刺代わりとなる連絡先データ（vCard）を書き込むことができます。
また、NFC非対応の端末（PCやiPhoneなど）でアクセスした場合でも、生成した連絡先データを .vcf ファイルとしてダウンロードし、端末の連絡帳に登録することが可能です。

✨ 主な機能 (Features)
連絡先情報の入力とプレビュー: フォームに情報を入力すると、リアルタイムにvCardデータが生成され、生データを確認できます。
データ容量チェック機能: NTAG215（504 Bytes）などの容量制限を意識し、入力データのバイト数をリアルタイムで計算・警告します。
NFCタグへの書き込み: Web NFC APIを使用し、ブラウザから直接NFCタグにvCardデータ（text/vcard）を書き込みます。
VCFファイルのダウンロード: 生成したvCardデータを .vcf ファイルとしてローカルに保存できます（全デバイス対応）。
フロントエンド完結: サーバーサイドの処理やビルド環境を必要とせず、ブラウザだけで安全に動作します。

📱 動作環境 (Requirements)
Web NFC APIは、セキュリティ上の理由から HTTPS環境（またはローカルの localhost）でのみ動作します。
NFC書き込み機能を利用する場合
対応OS: Android
対応ブラウザ: Google Chrome for Android（バージョン89以降）などのChromium系ブラウザ

デバイス要件:
端末のNFC機能が有効になっていること
NDEF（NFC Data Exchange Format）対応のNFCタグ（NTAG215, NTAG216 など推奨）
(※ iOSのSafariは現在Web NFC APIをサポートしていないため、NFCへの書き込み機能は利用できません)
VCFダウンロード機能のみを利用する場合
PC、Mac、iOS、Androidなど、ファイルのダウンロードが可能なモダンブラウザ全般で利用可能です。

🚀 使い方 (Usage)
アプリの操作手順
画面のフォームに「姓」「名」「電話番号」などの連絡先情報を入力します。
入力に合わせて「データ容量」ゲージが変化します。容量オーバーに気を付けてください。
Android端末の場合: 「NFCタグに書き込む」ボタンを押し、待機状態になったらスマートフォンの背面にNFCタグをぴったり重ねます。
PCやiPhoneなどの場合: 「VCFをダウンロード」ボタンを押して、生成されたファイルを保存し、連絡帳アプリで開いてください。

ローカルでの動作確認
ローカル環境でNFCのテストをする場合、file:// プロトコルではなくローカルサーバーを立ち上げる必要があります。
git clone [https://github.com/kmtshn/WebNFC_VCF.git](https://github.com/kmtshn/WebNFC_VCF.git)
cd WebNFC_VCF

# 例: Python3のローカルサーバー機能を使う場合
python -m http.server 8000
その後、同じネットワーク内のAndroid端末から http://<PCのローカルIP>:8000 にアクセスしてください。
